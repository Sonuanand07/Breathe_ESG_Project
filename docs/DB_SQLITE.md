# SQLite (Local Testing)

This app can run locally with SQLite for fast setup.

## Database Path

### Recommended DB file
- `backend/db.sqlite3`

### Settings environment variables
Set:

```env
DB_ENGINE=django.db.backends.sqlite3
DB_NAME=backend/db.sqlite3
```

## Local steps

```bash
cd backend
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

## Features
SQLite supports:
- Migrations (Django-managed)
- Transactions for ingestion + review updates
- Audit trail queries

## Trigger / Sequence
SQLite does not require PostgreSQL-style sequences/triggers for this project:
- Primary keys are UUID fields generated by Django/Python
- No custom DB triggers are required for correctness

If you later change primary key strategy, triggers/sequences might become relevant.
